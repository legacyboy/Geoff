# Top 100 Forensics Writeups - My Technical Blog

## Overview
A curated collection of forensic challenge write-ups and resources for CTF players.

## Featured Resources & Tools
- General Guides: Trail of Bits CTF Forensics guide.
- Memory Forensics: MemLabs (Abhiram Kumar), a dedicated introduction to memory forensics.
- File Analysis: Usage of binwalk, foremost, and exiftool for hidden data and file carving.
- Network Forensics: Analysis of PCAPs, DNS communication (dnscat2), and protocol analysis.

## Notable CTF mentions
- PicoCTF: Multiple write-ups for 2018 and 2019 forensics challenges.
- CTFLearn: Walkthroughs for Easy and Medium forensics tasks.
- csictf 2020: Challenges like 'Panda' and 'In Your Eyes'.
- InCTF Internationals 2020: Focus on memory and disk forensics (Investigation, Lookout Foxy).
- SANS Holiday Hack: Annual challenges focusing on diverse forensic artifacts.

## Techniques Highlighted
- Binary data manipulation with Python.
- Image steganography and recovery.
- Memory dump analysis using Volatility.
- Operating system artifacts (Windows Registry, NTFS).
