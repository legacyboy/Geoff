# AboutDFIR: Challenges & CTFs Overview

## Summary
A comprehensive curated list of DFIR-focused CTFs and challenges, ranging from 2004 to 2024.

## Key Challenge Categories
- Memory Forensics: 13Cubed Mini-Memory CTF, MemLabs, SANS Memory Challenge 2013, Pwn2Win CTF 2019.
- Network Analysis: Malware Traffic Analysis Exercises, SANS DFIR Monterey 2015, various PCAP-based challenges from PicoCTF and BCCTF.
- Disk & File Forensics: DFRWS challenges, Google CTF (Home Computer), various NTFS and corrupted file recovery tasks.
- Mobile Forensics: BelkaCTF #6 (iOS), Magnet Virtual Summit CTFs (iOS, Android), Cellebrite CTFs.
- Malware Analysis: FireEye FLARE-On (2014-2019), MalwareTech challenges.

## Notable Resource Links
- MemLabs: Educational CTF focusing on memory forensics.
- CyberDefenders: Various blue team challenges (e.g., GrabThePhisher).
- SANS Holiday Hack: Yearly miscellaneous forensic challenges.
- Cellebrite & Magnet Forensics: Specialized mobile and corporate forensic CTFs.

## Historical Timeline
The list tracks the evolution of forensic challenges from early insider threat cases (2004) and the DFRWS rodeo challenges (2005) to modern APT emulation labs and cloud-based forensic scenarios (2024).
