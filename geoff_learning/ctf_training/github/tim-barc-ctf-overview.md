# Tim-Barc CTF Writeups Overview

## Repository Summary
A comprehensive collection of CTF writeups covering various DFIR domains, primarily from CyberDefenders, TryHackMe, BTLO, and HackTheBox.

## Categories & Key Tools

### 1. Endpoint Forensics
- **Focus:** Investigating compromised Windows/Linux endpoints.
- **Key Tools:** FTK Imager, Registry Explorer, MFTECmd, Timeline Explorer, Volatility 3, Autopsy, PECmd.
- **Notable Labs:** RepoReaper (Hard), Maranhao (Easy), PwnedDC (Hard).

### 2. Network Forensics
- **Focus:** Packet analysis and log review.
- **Key Tools:** Wireshark, Zeek, Brim, NetworkMiner, Tshark.
- **Notable Labs:** Trident (Medium), NukeTheBrowser (Hard), Warzone 1 & 2 (Medium).

### 3. Mobile Forensics
- **Focus:** Android/iOS device analysis.
- **Key Tools:** ALEAPP, CLEAPP, jadx.
- **Notable Labs:** AndroidBreach (Medium), The Crime (Easy).

### 4. IDS/IPS & SIEM
- **Focus:** Threat detection using Snort, Splunk, ELK, and Wazuh.
- **Key Tools:** ELK Stack, Splunk, Wazuh, Snort.
- **Notable Labs:** HafinumAPT (Hard), Kerberoasted (Medium).

### 5. Cyber Threat Intelligence (CTI)
- **Focus:** Malware campaign tracking and attribution.
- **Key Tools:** VirusTotal, Malpedia, MITRE ATT&CK, Open CTI.

### 6. Email Analysis
- **Focus:** Phishing investigation, header analysis, and payload decoding.
- **Key Tools:** Thunderbird, mxtoolbox, URL2PNG, CyberChef.

### 7. Malware Analysis & Reverse Engineering
- **Focus:** Static and dynamic analysis of PE files and scripts.
- **Key Tools:** PE Studio, DIE, dnSpy, Any.Run, ProcMon, Capa, Floss.
- **Notable Labs:** AgentTesla (Medium), MalaCrypt (Medium).
