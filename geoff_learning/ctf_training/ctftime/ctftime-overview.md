# CTFtime Overview

## Description
CTFtime is the primary global hub for Capture The Flag (CTF) competitions, providing rankings, event schedules, and a repository of write-ups.

## Key Features
- **Global Rankings:** Tracks team performance across various years and regions.
- **Event Tracking:** Lists upcoming and past competitions (e.g., DawgCTF, UMassCTF).
- **Community Knowledge:** Serves as a central point for teams to publish and share their solve methodologies.
